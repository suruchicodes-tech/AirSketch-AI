AirSketch AI is an AI-powered virtual drawing application that allows users to draw in the air using hand gestures and computer vision technology. The system tracks finger movements in real time and converts them into digital sketches without requiring a physical pen or touch input.

Features ✨
Real-time hand gesture recognition
Virtual drawing using finger movements
Touchless interaction
Smooth and interactive user experience
AI and Computer Vision integration
Easy-to-use interface
Technologies Used 🛠️
Python
OpenCV
Computer Vision
Hand Tracking
AI Concepts
How It Works ⚡
The camera captures hand movements.
The system detects and tracks finger positions.
Finger gestures are translated into drawing actions.
Users can create sketches on a virtual canvas in real time.
