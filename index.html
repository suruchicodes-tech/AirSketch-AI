About This Project

AirSketch AI ✨

"Draw in the Air, Create Without Limits."

AirSketch AI is an innovative computer vision application that lets users draw in the air using hand gestures — no physical device, mouse, stylus, or touchscreen required.

Real-time hand tracking translates your finger movements into fluid digital strokes on a virtual canvas.

⚡ Key Features

🖐 Real-time hand gesture detection
🚫 Touchless virtual drawing
🎨 Multiple brush colors & sizes
🧹 Eraser & canvas clear tools
💾 Save & download as PNG
📱 Fully responsive design

🔬 Technologies Used

🐍 Python👁 OpenCV🖐 MediaPipe🔢 NumPy⚛️ React.js🌐 Flask☕ Spring Boot

🔄 How It Works

1
Webcam captures your live video feed in real time.
2
MediaPipe detects 21 hand landmarks per frame.
3
Index fingertip (landmark 8) position is extracted.
4
Coordinates map to canvas pixels for drawing strokes.
5
Gesture recognition switches between draw/erase/save.

🎯 Project Objective

AirSketch AI demonstrates the power of computer vision and human-computer interaction — showcasing real-world applications of AI, gesture recognition, and real-time image processing.
